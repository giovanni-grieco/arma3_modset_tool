# Arma 3 Preset Set Tool (client-side)

Simple client-side tool to load Arma 3 preset HTML files, perform set operations (union, intersection, difference), and export a new preset HTML file.

Usage:
- Open `index.html` in a modern browser.
- Load one or more Arma 3 preset HTML files using the file picker.
- Select presets (checkboxes), choose an operation, and click `Run`.
- Click `Export Result` to download a new preset HTML file.
